## Development Process

```
Discovery → Architecture Design → Prototyping → Sprint-based Development → Testing → Deployment → Handoff
```

1. **Discovery** (1–2 weeks): Requirements gathering, user research, tech stack finalization
2. **Architecture Design**: System design, database schema, API contracts
3. **Sprint Development**: 2-week sprints with demo at the end of each sprint
4. **QA**: Automated testing + manual testing before each release
5. **Deployment**: CI/CD pipeline setup, staging environment, production deployment
6. **Handoff**: Code documentation, knowledge transfer, post-launch support (30 days included)
